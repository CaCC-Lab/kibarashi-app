# backend/src/ 専用ルール

## 追加チェック

### API・外部連携
- 外部API呼び出しのタイムアウト設定
- リトライ処理の有無
- 接続エラーのハンドリング
- レート制限の考慮

### リソース管理
- データベースコネクションの解放
- メモリ使用量（大量データ処理時）
- ファイルハンドルのクローズ

### セキュリティ
- 入力バリデーション
- SQLインジェクション対策
- パストラバーサル対策
- CORS設定の適切性

## 禁止事項

- console.log の使用
- any型の使用
- bare catch
- ハードコードされた設定値・APIキー
